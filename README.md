# Blogging-website

<p> This is a website which is protected by authentication system which uses sessions and cookies using Passport.js. Here, users can create, edit as well as completely remove a post depending on their need. </p>

## Features
    1. User Registeration
    2. User Authentication & Authorization
    3. Compose/ Edit/ Delete POST

## Pages
    1. Home
    2. About us
    3. Compose
    4. Terms & Conditions
    5. Contact us

<p> Any of this subsection is only accessed when user has logged in to the website.  </p>

## Technologies used:
    1. ExpressJS (A NodeJS Framework)
    2. MongoDB (A NoSQL Database)
    3. PassportJS (For authentication & authorization)
    4. Md5 Hashing
    5. EJS (For Server side HTML rendering)
